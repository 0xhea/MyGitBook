## 数组

```go
var a1 [3]bool
```

### 初始化

```go
a1 := [4]int{1, 2, 3, 4}
a2 := [...]int{1, 2, 3, 4, 5, 6}  // 推断数组长度
a3 := [5]int{0: 1, 2: 3, 4}  // 根据索引值[1 0 3 4 0]
```

### 遍历

```go
for i := 0; i < len(a3); i++ {
    fmt.Println(a3[i])
}

for i, v := range a3 {
    fmt.Println(i, v)
}
```

### 多维数组

```go
a1 := [3][2]int{{1, 2}, {3, 4}, {5, 6}}
```

## 切片（slice）

引用类型

```go
var s1 []int
var s2 []int{1, 2, 3}
```

nil：空

### 由数组得到切片

```go
a1 := [...]int{1, 3, 5, 7, 9}
s1 := a1[0:4]  // 左闭右开
s1 := a1[1:]
s1 := a1[:4]
s1 := a1[:]
```
