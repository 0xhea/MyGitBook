# Vim

```
u  // undo
ctrl+r  // redo
v  // visual模式，可以进行选择
yy  // 复制
cc  // 剪切
p  // 粘贴
ctrl+n  // 自动补全，选项中只会有已经出现过的单词
ctrl+w  // 切换两个窗口的光标

:sp 上下分屏
:vs 左右分屏

zz  把当前行变为中心点
```
